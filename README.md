# CHIPPU-hachi
CHIP-8 emulator

https://en.wikipedia.org/wiki/CHIP-8

## Building

#### SDL2 backend
```
cmake -DSDL2_BACKEND=ON .
make
```

#### Ncurses backend
```
cmake -DNCURSES_BACKEND=ON .
make
```
